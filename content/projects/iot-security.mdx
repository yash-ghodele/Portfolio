# IoT Smart Lock: Next-Gen Home Security

Combining robust hardware with smart connectivity for ultimate peace of mind.

## Features
-   **Multi-Auth:** Fingerprint + PIN + App Unlock.
-   **Intruder Alert:** 3 incorrect attempts triggers a lockdown and photo capture.
-   **Remote Access:** Control your door from anywhere via the Blynk app.

## Hardware Design
Custom PCB design integrating the ESP8266 Wi-Fi module, capacitive fingerprint sensor, and solenoid lock mechanism.
