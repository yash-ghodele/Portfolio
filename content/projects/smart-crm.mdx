### VBA & WinAPI
We used **Variant Arrays** to treat Excel sheets like database tables.

```vb
' 1. Optimization: Read 100k rows into RAM
Dim DataArray As Variant
DataArray = Range("A1:Z100000").Value

' 2. Process in Memory (Millisecond speed)
' Avoiding interaction with the Excel UI layer entirely
For i = LBound(DataArray) To UBound(DataArray)
    If IsValidEmail(DataArray(i, 5)) Then
        DataArray(i, 6) = "Verified"
    End If
Next i

' 3. Dump back to Sheet (Single Atomic Op)
Range("A1:Z100000").Value = DataArray
```

### Reliability
To simulate **Transactional Safety** in VBA:
*   **Auto-Backup:** Shell scripts trigger timestamped `.accdb` backups before every run.
*   **Error Trapping:** Custom error handlers log issues without crashing the application loop.
