# Smart CRM: Automating Growth

A bespoke CRM solution designed for high-volume B2B lead processing.

## The Workflow
1.  **Ingest:** Auto-import leads from CSV/Excel sources.
2.  **Verify:** 5-step validation (Syntax, Domain, SMTP check, Duplicate check, Blacklist).
3.  **Enrich:** Add company details via API lookups.
4.  **Export:** Clean datasets ready for cold outreach.

## Technical Highlights
Built with robust **VBA automation** and **MS Access** for a cost-effective, high-performance database solution handling 100k+ records.
