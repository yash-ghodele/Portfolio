## Carousel Functionality

### Display Format
- **Visible Cards:** 3 at a time
- **Layout:** Grid (1 column mobile, 3 columns desktop)
- **Rotation:** Continuous wrap-around

### Navigation Options

#### Desktop
1. **Arrow Buttons**
   - Left arrow: Previous set
   - Right arrow: Next set
   - Positioned outside grid

2. **Dot Indicators**
   - Bottom center
   - One dot per testimonial
   - Click to jump to specific position

#### Mobile
- Same navigation options
- Single column display
- Swipe-friendly (future enhancement)

## Card Design

### Visual Elements
1. **Quote Icon**
   - Top right corner
   - Low opacity background
   - Increases opacity on hover

2. **Star Rating**
   - 5 gold stars (always full)

3. **Avatar Initial**
   - Circular gradient background

   - Border glow on hover
